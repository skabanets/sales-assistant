# Sales Assistant

## Submodules

This project uses Git submodules for managing dependencies. Below are the instructions for adding
and updating submodules.

### Adding a Submodule

To add a new submodule to the project, run the following command:

```
git submodule add https://gitlab.com/public-stud-prog/public-common src/interfaces-submodule
```

### Initializing and Updating Submodules

After adding a submodule, you can initialize and update it using the following command:

```
git submodule update --init --recursive
```
